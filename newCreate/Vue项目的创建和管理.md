## Vite脚手架基本使用

(一)npm安装项目
```
$ npm init  @vitejs/app  <project-name>  --template vue
$ cd <project-name>
$ npm install
$ npm run dev
```

(二)yarn 创建项目
```
$ yarn create @vitejs/app <project-name>  
$ cd <project-name>
$ yarn
$ yarn dev
```
注: 内部可选运行那种类型的项目
